Personal Website - The Blog

This a resource page to document and understand personal blogging website.

Code repositories Environments

Code repositories

blog

This backend service exposes APIs for creating, updating, reading, and deleting blogs.
auth

This backend service exposes APIs for user signup and signin.
registry

This backend service works as service registry for blogging backend services.
backend-devops

This repository is used to handle devops tasks for blogging backend services.
frontend

This is the frontend for blogging website.

Environment Information

Backend Application Services - Host addresses

blog-app-server: http://localhost:8081
auth-app-server: http://localhost:8080
service-registry-server: http://localhost:8761

Databases Connection URLs

blog-db-server: mongodb://[[username]]:[[password]]@localhost:27017/blogDb
auth-db-server: jdbc:mysql://[[username]]:[[password]]@localhost:3306/authDb?allowPublicKeyRetrieval=true&autoReconnect=true&useSSL=false&verifyServerCertificate=false
